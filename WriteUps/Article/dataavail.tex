The datasets analyzed in the current study are available in the MATPOWER repository \cite{matpower} at \href{https://matpower.org/}{https://matpower.org/} \add{and the NESTA repository \cite{CoGoSc2019}, the instances of which are available as part of the Library of IEEE PES Power Grid Benchmark repository at}

No new datasets were created as part of this study.
